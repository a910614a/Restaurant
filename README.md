# 🍴 Restaurant 網頁專案

## 專案簡介
這是一個餐廳網站的練習專案，提供使用者瀏覽餐廳資訊、了解品牌故事、查看門市位置，以及回報問題的功能。  
專案使用 **HTML、CSS、JavaScript** 建構，並透過 **GitHub Pages** 部署。

## 功能特色
- **首頁 (index.html)**：餐廳主頁面，展示品牌形象與主要資訊  
- **關於我們 (aboutus.html)**：介紹餐廳理念與背景  
- **門市資訊 (store.html)**：提供各分店的照片與位置  
- **常見問題 (problem.html)**：使用者可查看常見問題
- **聯絡我們 ()**：使用者可透過連結聯繫店家
- **樣式 (style.css)**：統一網站的設計風格  
- **互動 (js/)**：基本的 JavaScript 功能與互動效果  

## 技術使用
- **HTML5**：結構化網頁內容  
- **CSS3**：排版與樣式設計  
- **JavaScript**：互動功能  
- **GitHub Pages**：網站部署與展示
